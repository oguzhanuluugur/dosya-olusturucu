# Dosya Oluşturucu

Tek dosyalık offline web uygulaması ile kolayca dosya oluşturun ve indirin. WhatsApp/Mail uyumlu PDF ve ZIP çıktıları üretin.

## Özellikler

- **Çoklu Format Desteği**: TXT, HTML, JSON, PDF, ZIP
- **PDF Üretimi**: jsPDF ile otomatik sayfa kırılması
- **ZIP Paketleme**: JSZip ile klasör paketleme
- **Klasör Seçimi**: Masaüstünden klasör seçip ZIP'e çevirme
- **WhatsApp/Mail Uyumlu**: Tek dosya çıktıları
- **Offline Çalışır**: CDN kütüphaneleri ile tek dosya
- **Responsive Tasarım**: Modern UI

## Kullanım

1. Tarayıcınızda `index.html` dosyasını açın
2. Şablon seçin (Not, Rapor, Fatura, Dilekçe)
3. Başlık ve içerik girin
4. Çıktı türü seçin (PDF/ZIP önerilir)
5. "Dosyayı İndir" butonuna tıklayın

### Klasör Paketleme

- "Klasör Aç" butonuna tıklayın
- Masaüstünden klasör seçin
- "Klasörü ZIP'e İndir" ile paketleyin

## Kurulum

Dosyaları indirin ve `index.html`'i tarayıcıda açın. İnternet bağlantısı gerektirmez (CDN'ler yüklü).

## Teknolojiler

- HTML5, CSS3, JavaScript
- jsPDF (PDF üretimi)
- JSZip (ZIP paketleme)
- Unsplash API (logo için)

## Güvenlik

- `.env` dosyası hassas bilgileri içerir
- GitHub'a push etmeyin
- API anahtarlarını koruyun
